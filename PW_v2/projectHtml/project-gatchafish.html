March 7th 2023 [Personal Project / For Fun]

gat-cha fish.

A game inspired by the idle-game category at the time.
My main vision for this project was a game you'd put on the background, and when you heard a little ping that's when you'd know you caught a fish.
You'd quickly go back to the game and press the correct sequence of buttons to reel the fish in. The goal was to simulate a more realistic fishing experience.

Unfortunately people didn't understand the initial concept of it and this was recieved poorly because people thought it was a a very-active game. When it was actually the opposite.
Which taught me how I need to make these game explanations a lot more clearer, as looking back I can definiltey understand their perspective.

- THERE IS NO SAVE STATE FOR THIS GAME
- Game can only be played on 760px or wider
